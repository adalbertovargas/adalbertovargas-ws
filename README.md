## The content

Within the download you'll find the following directories (and files), logically grouping common assets. You'll see something like this:

```
├───info
│   └───preview
└───public
    ├───assets
    │   ├───css
    │   ├───fonts
    │   ├───images
    │   └───js
    ├───cloud
    └───seq         => seQ App
```

Sometimes is compiled and minified CSS and JS (`file.min.*`).
